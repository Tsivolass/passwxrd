Η εφαρμογή αυτή είναι σχεδιασμενη για να μπορεις να διαχειρίζεσαι ολους σου τους κωδικούς σε όλα τα chromium based browsers μεσω μιας εφαρμογης, και να προστατευει τους κωδικούς από επιθεσεις που γινονται μεσω κακοβουλων λογισμικών. το πρόγραμμα είναι ολο γραμμενο σε python, είναι open source ( GitHub ) και είναι offline. Χρησιμοποιει base64 encryption για να μετατρεψει το master password σου σε ένα token, το οποιο θα χρησιμοποιηθεί για να γινει encrypt το vault του passwxrd, ενώ η κανονικη database των chromium-based browsers γινεται clear.
